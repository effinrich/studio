# **App Name**: TenantEase

## Core Features:

- Tenant Directory: Maintain a searchable list of all tenants, including contact info and lease terms.
- Rent Tracker: Log rent payments with date, amount, and method, with balance calculations. Late payments are automatically highlighted and landlord can use a tool that will make decision about the need of messaging the tenant.
- Automated Reminder Messaging: Set automated messages or message templates to be sent by email or SMS to tenants for various scenarios like rent dues and notifications.
- Document Management: Store and organize lease agreements, application forms, and other important documents for each tenant.
- Expense Tracker: Track property-related expenses (repairs, maintenance, etc.) and assign them to specific tenants or properties for accurate profitability analysis.
- Issue Reporting: Tenants can report issues/requests, which will be sent to the landlord to organize, categorize, prioritize and track issues through resolution.
- Lease Expiration Alert: The tool generates notifications when a lease is about to expire, to help plan renewals and keep up with other alerts related to deadlines.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6), conveying trust and stability, which are ideal for a financial app.
- Background color: Very light blue (#E3F2FD), providing a calm, unobtrusive backdrop.
- Accent color: Muted green (#81C784), offering a subtle, reassuring accent for key actions and highlights.
- Font pairing: 'Poppins' (sans-serif) for headings, providing a modern and precise look, with 'PT Sans' (sans-serif) for body text.
- Use clear, outline-style icons from a consistent set (e.g., FontAwesome or Material Icons) to represent key actions and data categories.
- Employ a clean, card-based layout to organize information. Prioritize clarity and readability, making it easy to quickly find key data. Maintain high data density, enabling use on many properties and tenants.
- Incorporate subtle transitions and animations (e.g., modals appearing, accordions expanding) to create a smooth and engaging user experience.
